# Boerenkost

## How to run?

Run following script for dev profile:

	./runDev.sh
		
Tomcat will start on port 8081.

## Accessing H2 database

Go to:

	http://localhost:8081/h2-console
	
	
And login with admin - admin. Check that JDBC URL is 

	jdbc:h2:mem:testdb
